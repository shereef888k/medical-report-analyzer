# Medical Report Analyzer (AI)

## Overview
This project is an AI-powered system that analyzes medical reports (PDF/images) and extracts useful information such as diagnosis, patient details, and health insights. It helps users quickly understand complex medical data.

## Features
- Upload medical reports (PDF/image)
- Extract key medical information
- AI-based report analysis
- Generates simplified insights
- User-friendly interface

## Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript
- AI/LLM integration

## How it Works
1. User uploads medical report
2. System extracts text from document
3. AI analyzes the extracted data
4. Provides structured output and insights
